# Extract ordinal diagnostics from a reliability() result

Extract ordinal diagnostics from a reliability() result

## Usage

``` r
ordinal_diagnostics(x)
```

## Arguments

- x:

  An object returned by
  [`reliability()`](https://winzarh.github.io/LikertMakeR/reference/reliability.md).

## Value

A data.frame describing observed response categories and sparsity
checks, or NULL if no diagnostics are available.
